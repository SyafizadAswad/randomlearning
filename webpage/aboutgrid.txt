Key Techniques Used:
Overlay: The .background-overlay div ensures the white text remains readable against the mountain background.

Gap & Padding: The layout relies on heavy padding (40px) inside the cards and a wide gap (30px) between grid items to get that "airy" professional feel.

Skill Bars: I used simple nested div elements with a background and a specific width percentage to mimic the progress bars in your screenshot.

Flexibility: While the image looks a bit like masonry (where heights vary), CSS Grid handles this naturally by aligning cards to the top of their row. If you want the bottom row to "tuck in" higher, you would use grid-row: span 2 on larger items.

===============================

To achieve this specific layout where box sizes change row-by-row, we move away from simple column definitions and use a 12-column grid system.A 12-column grid is the "magic number" in web design because it is divisible by 2, 3, 4, and 6. This allows us to easily calculate your requested fractions:Equal size: 6 columns each ($6 + 6 = 12$)3/4 size: 9 columns ($9 + 3 = 12$)The Grid LogicWe will use the grid-column property to tell each box exactly how many of those 12 tracks to span.CSS.grid-container {
    display: grid;
    /* Create 12 equal-width columns */
    grid-template-columns: repeat(12, 1fr); 
    gap: 20px;
    max-width: 1000px;
    margin: 0 auto;
}

/* Row 1: Equal sizes (6 columns each) */
.profile-text { grid-column: span 6; }
.skills       { grid-column: span 6; }

/* Row 2: First box takes 3/4 (9 columns) */
.education   { grid-column: span 9; }
.languages   { grid-column: span 3; }

/* Row 3: Second box takes 3/4 (9 columns) */
.contact     { grid-column: span 3; }
.experiences { grid-column: span 9; }
How to Visualize the TracksBy defining grid-template-columns: repeat(12, 1fr), you are drawing 11 vertical lines on your page. When you say span 9, the browser stretches that box across 9 of those 12 available "slices."RowBox 1 SpanBox 2 SpanLogicRow 1span 6span 6$6/12 = 1/2$ (Equal)Row 2span 9span 3$9/12 = 3/4$Row 3span 3span 9$9/12 = 3/4$ (on the right)Pro-Tip: ResponsivenessOn a mobile device, a 12-column span of 3 (the $1/4$ size boxes) will look very squished. You should add a media query to make every box span the full width on smaller screens:CSS@media (max-width: 768px) {
    .card {
        grid-column: span 12 !important;
    }
}
This keeps your layout "zigzagged" and professional on your desktop monitor while ensuring it remains readable for someone checking your profile on their phone in Okayama.
