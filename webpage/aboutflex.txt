First: The Mental Model (THIS is where most people fail)

Flexbox is 1-dimensional layout control.

👉 You always have:

Main axis (direction items flow)
Cross axis (perpendicular direction)
display: flex;
flex-direction: row;   /* default */
Main axis → horizontal →
Cross axis → vertical ↓

If you switch:

flex-direction: column;
Main axis ↓ vertical
Cross axis → horizontal

🔥 Rule #1: Always ask → “Which axis am I controlling?”

🎯 The 3 Most Important Properties (90% of use cases)
1. justify-content → controls MAIN axis
2. align-items → controls CROSS axis
3. gap → spacing between items
Example (center everything)
6
.container {
  display: flex;
  justify-content: center; /* main axis */
  align-items: center;     /* cross axis */
  height: 100vh;
}
📦 Flex Container vs Flex Item
Container (parent)

Controls layout:

display: flex;
flex-direction
justify-content
align-items
gap
Item (child)

Controls behavior:

flex-grow
flex-shrink
flex-basis
align-self
🔥 The “Holy Trinity” for Items
1. flex-grow

Who expands?

.item {
  flex-grow: 1;
}

If 3 items all have flex-grow: 1 → equal width

2. flex-basis

Starting size

.item {
  flex-basis: 200px;
}
3. Shortcut: flex
flex: 1;

Means:

flex-grow: 1;
flex-shrink: 1;
flex-basis: 0;

👉 This is why things magically “fill space”

🧩 Common Layout Patterns (You’ll use these a LOT)
✅ 1. Horizontal layout (navbar)
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
✅ 2. Equal columns
.item {
  flex: 1;
}
✅ 3. One side fixed, one flexible
.sidebar {
  width: 250px;
}

.content {
  flex: 1;
}
✅ 4. Push item to the right
.item {
  margin-left: auto;
}

🔥 This is a pro trick most beginners don’t know.

✅ 5. Vertical stacking
.container {
  display: flex;
  flex-direction: column;
}
🧠 Why you keep “failing” with Flexbox

Let me guess your issues:

❌ “align-items not working”

👉 You forgot height

.container {
  height: 100px; /* needed */
}
❌ “justify-content not doing anything”

👉 No extra space exists

Flex only distributes remaining space

❌ “things not centered”

👉 You mixed up axis

❌ “items overflow weirdly”

👉 Default:

flex-shrink: 1;

Things will shrink unless you stop it:

flex-shrink: 0;
🧾 Mini Cheat Sheet
/* Container */
display: flex;
flex-direction: row | column;

justify-content: 
  flex-start | center | flex-end | space-between | space-around;

align-items:
  stretch | flex-start | center | flex-end;

/* Item */
flex: 1;
align-self: center;
margin-left: auto;

alignment issues
